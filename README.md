# CreditWise Loan Approval System

This is a simple Machine Learning project that predicts loan approval using applicant data.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Model
- Logistic Regression
- Linear Regression

## Files
- Untitled.ipynb
- loan_approval_data.csv
- loan_model.pkl

## Output
Predicts whether a loan is Approved or Not Approved.
